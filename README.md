# lyh-uniappx

一个基于 uni-app x 开发的多平台移动应用项目。

## 项目简介

这是一个使用 uni-app x 框架开发的跨平台移动应用，支持 Android、iOS、微信小程序等多个平台。项目采用 Vue 3 语法，使用 TypeScript 进行开发。

## 项目结构

```json
lyh-uniappx/
- App.uvue # 应用入口文件
- main.uts # 主程序文件
- manifest.json # 应用配置文件
- pages.json # 页面路由配置
- uni.scss # 全局样式文件
- components/ # 组件目录
- pages/ # 页面目录
- static/ # 静态资源
```

## 功能特性

- **多平台支持**: 支持 Android、iOS、微信小程序、支付宝小程序、百度小程序、字节跳动小程序等
- **自定义导航栏**: 使用自定义底部导航栏组件
- **响应式设计**: 适配不同屏幕尺寸
- **TypeScript 支持**: 使用 UTS 语言进行类型安全的开发

## 页面说明

- **首页 (Home)**: 应用主页面，显示欢迎信息和应用图标
- **发现页 (Discover)**: 内容发现页面
- **钱包页 (Wallet)**: 钱包功能页面

## 开发环境

- uni-app x 框架
- Vue 3
- TypeScript (UTS)
- 支持多端编译

## 运行项目

1. 安装 HBuilderX 开发工具
2. 打开项目根目录
3. 选择运行到相应平台

## 构建说明

项目使用 uni-app x 进行构建，支持以下平台：

- App (Android/iOS)
- 微信小程序
- 支付宝小程序
- 百度小程序
- 字节跳动小程序
- Web
